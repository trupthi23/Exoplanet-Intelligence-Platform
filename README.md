# Exoplanet Intelligence Platform

## Overview

A full-stack data science platform that analyzes NASA exoplanet data, predicts planetary habitability, calculates Earth Similarity Index (ESI), and provides interactive analytics dashboards.

## Tech Stack

- Python
- FastAPI
- React
- PostgreSQL
- Scikit-learn
- Plotly
- SQLAlchemy
- GitHub

## Features

- Planet Explorer
- Discovery Analytics
- Earth Similarity Score
- Habitability Prediction
- Interactive Dashboard
